# Loader Mini Game 🎮

A tiny fun "Catch the Bug" game designed to replace boring loaders in dev environments.

⚠️ Not for production use (unless you're brave 😄)

## Features
- Simple interactive UI
- Stress-relief micro game
- Pure HTML + JS

## Usage
Open index.html in browser and play!

## Idea
Built as a fun way to reduce developer stress during loading time.
